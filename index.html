INSAConnect

Extension Google Chrome

INSAConnect est une extension Google Chrome développée par mes soins. L'objectif de cette extension est d'offrir aux élèves et enseignants de l'INSA une connection automatique au réseau de l'INSA via le wifi dans les bâtiments ainsi que dans les résidences Promologis situées sur le campus.

La connexion habituelle passe par une page captive ouverte à la connexion au Wifi où l'utilisateur rentre son identifiant et son mot de passe pour se connecter puis clique sur connecter pour établir la connexion. Cette opération est faite par mon extension à partir des identifiants fournis par l'utilisateur à l'extension

Tout l'intérêt de l'extension est la sauvegarde permanente des identifiants dans le local storage. Pour protéger le mot de passe utilisateur un système de cryptage est mis en place pour empêcher la récupération malveillante de cette information.

RioSport

Application Android avec Web Service

RioSport est une application Android développée en Java au cours de mon semestre d'étude au Brésil. Nous avons été 3 a réaliser ce projet. Il s'agissait de créer un réseau social autour du sport. L'application permet de mettre en contact des utilisateurs désirant se rencontrer pour pratiquer leur sport ensemble.

Ce projet comporte la création de 4 modules complémentaires. Il se compose d'un client Android développé en Java avec le logiciel Netbeans, d'un serveur PHP qui interroge une base de donnée MySQL ainsi que d'un Web Service faisant le lien entre le serveur et le client en transmettant les informations au format XML.

Chaque utilisateur est enregistré dans notre base de donnée mais sa connexion à notre application se fait à travers la connexion à son compte Google Plus. Il doit donc posséder un compte Google Plus pour pouvoir utiliser l'application ce que la plupart des utilisateurs d'un smartphone Android possèdent. Nous avons donc utilisé l'API Google+ dans notre application.

Une fois connecté, l'utilisateur peut publier des évènements sportifs à venir ou consulter ceux publiés par ses amis ou ceux qui sont publics afin d'y participer.

Ce projet a été réalisé dans le cadre du cours Software para Smartphone suivi à l'UFRJ avec le professeur Sergio Barbosa Villas-Boas (www.sbvb.com.br/).

MovieFinder

Application Windows

MovieFinder est un logiciel pour Windows capable d'analyser les films présents sur le disque dur de l'utilisateur et de les organiser dans une bibliothèque intuitive.

A partir du nom du film, une requête au site web Allociné est envoyé. C'est la page de recherche qui est visée. La page contient 5 films avec leur nom et leur date de réalisation pouvant correspondre au film de l'utilisateur. Une analyse du contenu html de la page reçu permet de récupérer ces élements et de les présenter à l'utilisateur. Celui-ci choisit celui qui correspond effectivement au film qu'il possède. Une nouvelle requête est ensuite de nouveau envoyé à Allociné vers la page exacte de ce film. Une dernière analyse de la page html permet de récupérer la date de réalisation, les 3 acteurs principaux, les 3 réalisateurs principaux, le genre, la note sur 5 des utilisateurs du site et l'affiche. Ces informations sont stockées dans la base de donnée contruite en PostgreSQL.

La dernière information contenue dans la base de donnée est si le film a déjà été visionné par l'utilisateur ou non. C'est là que l'application prend tout son sens. Un soir, un utilisateur peut avoir envie de regarder un film où Brad Pitt joue et qu'il n'a pas encore visonné ou bien encore tous les films d'action datant d'avant 2000. Il n'a qu'à entrer ces critères dans la bibliothèque et une liste de tels films lui est proposé. Il n'a plus alors qu'à choisir le film en fonction de sa note sur Allociné (qu'il peut modifier à tout moment), ou bien du réalisateur et cliquer sur lecture.

Java PostgreSQL
